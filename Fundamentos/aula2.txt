Strings:

- No JavaScript temos bibliotecas e metodos nativos para trabalhar com as variáveis dinâmicamente tipadas
.replace("Y","y") //troca uma caractere por outro

// Concatenação:
- Não usar o "+" para concatenar um texto com variáveis, existe a possibilidade do resultado ser diferente deivido a tipagem dinâmica

números:

- .isIntinger //verifica se um número é inteiro ou não inteiro
- (true/false)
- .toFixed(2) //formartar em 2  casas decimais

- NaN (Not a Number)

- O JavaScript automaticamente determina se uma variável do tipo numérica não inteira precisa de alta precisão

ATENÇÃO!!!

- Obsoleto:
- var //criação de variáveis
- == // igual a
- != // diferente de

- substituir por:
- let // criação de variáveis
- === // idêntico a 
- !== // não idêntico a 

ATENÇÃO!!!

- No JavaScript podemos omitir as chaves nas estruturas de controle, porém só uma linha de código é processada (CUIDADO!!!)
